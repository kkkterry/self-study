49강. 클래스
- 접근제한 지정자(private, protected(상속), public)
  : 클래스 내의 맴버변수 or 맴버변수의 접근 레벨
- 생성자: 클래스이름()
   : 반환타입이 없다.
   : 자동으로 호출이 된다.
   : 이니셜라이져(생성자 특수 문법)
   : 선언하지 않아도 자동으로 생성된다.
- 소멸자: ~클래스이름()
   : 클래스가 없어질때 자동으로 호출된다.
   : 선언하지 않아도 자동으로 생성된다.
- 맴버함수 : 클래스가 사용하는 전용함수
              : 클래스의 객체가 필요함(this(객체의 주소) 생략되어있음)
              : 맴버함수를 객체를 통해서만 호출하면, 해당 객체의 주소가 this 포인터로 전달된다.

* 래퍼런스 : int& iRef = a;
	 : 원본을 참조한다 
	 : 참조를 통해서 직접적으로 원본을 수정 가능
	 : int& iRef = a; , iRef = 100;
