48-50강. 연결형리스트

연결형리스트: 첫번째 노드 주소 + 리스트안 노드의 수
노드: 데이터 + 다음 노드 주소
-> 첫번째 주소만 알면 다른 노드 접근이 가능

- LinkedList.h
  - 노드 구조체(데이터, 다음 노드 주소)
  - 연결리스트 구조체(노드 구조체, 리스트안 노드수)
  - 연결형 리스트 초기화
  - 데이터 입력
  - 데이터 앞부분에 추가
  - 리스트 메모리 해제

- LinkedList.cpp
  - 연결형 리스트 초기화 구현
  - 데이터 입력 구현
  - 데이터 앞부분에 추가 구현
  - 리스트 메모리 해제 구현

- LinkedListM.cpp
  - 연결형 리스트 선언
  - 연결형 리스트 초기화
  - 데이터 입력
  - 리스트 메모리 해제
