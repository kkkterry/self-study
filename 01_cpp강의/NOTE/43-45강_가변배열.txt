43-45강. 가변배열
: 프로그램 실행도중 프로그램의 필요에 따라 증감하는 배열

1. int a = 10;
int arr[a] ={};
위와 같이 선언하면 안된다.
-> 개수가 정해지지 않은 배열은 지역에다 할당 할 수 없다.
-> 가변배열
    : Arr.h
      - 구조체(데이터주소, 데이터수, 데이터 저장 갯수)
      - 초기화 함수
      - 공간추가 함수
      - 데이터 추가 함수
      - 데이터 메모리 해제 함수

    : Arr.cpp
      - 함수 구체적으로 구현

    : main
      - 구조체 선언하고 최기화 하여 사용
